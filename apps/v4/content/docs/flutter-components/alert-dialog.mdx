## Usage

```dart
import 'package:your_app/components/ui/alert_dialog.dart';
```

```dart
FlutterAlertDialog(
  trigger: ElevatedButton(
    onPressed: () {},
    child: const Text('Show Alert'),
  ),
  content: AlertDialogContent(
    title: const AlertDialogTitle(
      child: Text('Are you absolutely sure?'),
    ),
    description: const AlertDialogDescription(
      child: Text(
        'This action cannot be undone. This will permanently delete your '
        'account and remove your data from our servers.',
      ),
    ),
    actions: Builder(
      builder: (context) => AlertDialogFooter(
        children: [
          AlertDialogCancel(
            onPressed: () => Navigator.of(context).pop(),
            child: const Text('Cancel'),
          ),
          AlertDialogAction(
            onPressed: () {
              // Perform your action here
              Navigator.of(context).pop();
            },
            child: const Text('Continue'),
          ),
        ],
      ),
    ),
  ),
)
```

### Custom Styled Dialog

```dart
FlutterAlertDialog(
  barrierDismissible: false,
  barrierColor: Colors.black.withOpacity(0.7),
  trigger: FilledButton.icon(
    onPressed: () {},
    icon: const Icon(Icons.warning),
    label: const Text('Show Warning'),
  ),
  content: AlertDialogContent(
    maxWidth: 400,
    padding: const EdgeInsets.all(32),
    gap: 24,
    title: const AlertDialogTitle(
      child: Row(
        children: [
          Icon(Icons.warning_amber, color: Colors.orange),
          SizedBox(width: 8),
          Text('Warning'),
        ],
      ),
    ),
    description: const AlertDialogDescription(
      child: Text('This is a warning message that requires your attention.'),
    ),
    actions: Builder(
      builder: (context) => AlertDialogFooter(
        children: [
          AlertDialogCancel(
            onPressed: () => Navigator.of(context).pop(),
            child: const Text('Dismiss'),
          ),
          AlertDialogAction(
            onPressed: () => Navigator.of(context).pop(),
            child: const Text('Acknowledge'),
          ),
        ],
      ),
    ),
  ),
)
```

### Dialog with Custom Content

```dart
FlutterAlertDialog(
  trigger: OutlinedButton(
    onPressed: () {},
    child: const Text('Custom Dialog'),

  content: AlertDialogContent(
    children: [
      const Text(

      const TextField(
        decoration: InputDecoration(

      ),
      const SizedBox(height: 16),
      Builder(
        builder: (context) => Row(
          mainAxisAlignment: MainAxisAlignment.end,
          children: [
            TextButton(
              onPressed: () => Navigator.of(context).pop(),
              child: const Text('Cancel'),
            ),
            const SizedBox(width: 8),
            FilledButton(
              onPressed: () => Navigator.of(context).pop(),
              child: const Text('Submit'),
            ),
          ],
        ),
      ),
    ],
  ),
)
```

## Component API

### FlutterAlertDialog

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `trigger` | `Widget` | required | Widget that triggers the dialog when tapped |
| `content` | `Widget` | required | The content of the dialog |
| `barrierDismissible` | `bool` | `true` | Whether tapping outside dismisses the dialog |
| `barrierColor` | `Color?` | `Colors.black.withOpacity(0.5)` | Color of the modal barrier |
| `onOpenChanged` | `ValueChanged<bool>?` | `null` | Callback when dialog open state changes |

### AlertDialogContent

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `title` | `Widget?` | `null` | Dialog title widget |
| `description` | `Widget?` | `null` | Dialog description widget |
| `actions` | `Widget?` | `null` | Action buttons (typically Cancel and Confirm) |
| `maxWidth` | `double` | `512.0` | Maximum width of the dialog |
| `padding` | `EdgeInsets?` | `EdgeInsets.all(24)` | Content padding |
| `gap` | `double` | `16.0` | Gap between elements |
| `children` | `List<Widget>?` | `null` | Custom children (alternative to title/description/actions) |

### AlertDialogFooter

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `children` | `List<Widget>` | required | Action buttons |
| `gap` | `double` | `8.0` | Gap between buttons |
| `reverseOnMobile` | `bool` | `true` | Reverse button order on mobile (< 640px width) |

### AlertDialogAction

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `onPressed` | `VoidCallback?` | required | Callback when button is pressed |
| `child` | `Widget` | required | Button content |
| `autofocus` | `bool` | `false` | Whether to autofocus this button |

### AlertDialogCancel

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `onPressed` | `VoidCallback?` | required | Callback when button is pressed |
| `child` | `Widget` | required | Button content |
| `autofocus` | `bool` | `false` | Whether to autofocus this button |

## Accessibility

The Alert Dialog component follows Material 3 accessibility guidelines:

- **Keyboard Navigation**: Escape key dismisses the dialog
- **Focus Management**: Focus is trapped within the dialog when open and returns to trigger when closed
- **Screen Readers**: Proper semantic labels and announcements
- **Touch Targets**: Minimum 48x48dp touch targets for buttons
- **Color Contrast**: WCAG 2.2 AA compliant color combinations

## Design Specifications

Based on shadcn/ui and Material 3:

- **Overlay**: Black with 50% opacity, full screen
- **Content**: Max width 512px (mobile: 100% - 32px margin)
- **Border Radius**: 8px
- **Padding**: 24px
- **Shadow**: Material 3 elevation level 3 (24dp)
- **Animations**:
  - Fade in/out: 0 → 1 opacity
  - Scale: 95% → 100%
  - Duration: 200ms
  - Curve: Ease out
- **Typography**:
  - Title: 18px, semibold (600)
  - Description: 14px, regular (400), muted color
- **Layout**:
  - Mobile: Centered text, stacked buttons (reversed)
  - Desktop: Left-aligned text, inline buttons (right-aligned)

## Notes

- The component uses Material 3 design by default

- Responsive layout adapts to screen size (breakpoint: 640px)
- Focus management ensures keyboard accessibility
